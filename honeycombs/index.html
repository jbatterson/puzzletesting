How to Play

Fill every empty hexagon with a number so that the complete sequence 1 → N traces a connected path — each consecutive pair of numbers must share an edge.

After you place a number, the next one highlights on the keyboard. Tap an empty cell beside that previous number to place it in one tap. Tap any other empty cell (or a filled cell) to select it and clear the highlight — then pick a number or continue from there.

Tap the header, a clue hex, or anywhere outside the grid or keyboard to clear selection and the blue highlight—or tap the selected hex or highlighted number again. Tap any dark number to choose the active value, or a gray number after selecting a cell to move that value (the old cell empties and the replaced number returns to the keyboard).

If a hex is selected (blue ring), number keys type into that hex. Use the backspace key on the keyboard row to clear that hex, or Undo to step back through moves. Press Escape to clear selection so a number key picks the blue keyboard “pen” for tap-to-place on empties.

When all cells are filled, use the Check button below the keyboard to trace the path from 1 through consecutive numbers. A green line means a valid path; red means the path got stuck. The line fades after a few seconds or when you interact.
